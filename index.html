❓ Bantuan & Tutorial

Mulai:

  1. Klik "Mulai AR" untuk memulai aplikasi
  2. Arahkan kamera ke gambar marker
  3. Klik model 3D untuk melihat informasi detail
  4. Gunakan tombol kontrol untuk memutar dan memperbesar

Kontrol Gerakan:

  • Geser Kiri/Kanan: Memutar model
  • Cubitan: Memperbesar / memperkecil
  • Ketuk dua kali: Atur ulang tampilan

Tips:

  • Gunakan mode layar penuh untuk pengalaman lebih baik
  • Aktifkan rotasi otomatis di pengaturan
  • Periksa penghitung FPS untuk performa
